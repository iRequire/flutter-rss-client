# Flutter RSS Client
A simple RSS Reader made with Flutter

## Current Features
- Adding feeds by URL
- Reordering feeds by dragging
- Deleting feeds by swiping
- Viewing feed articles
- Dark/Light themes

## Planned Features
- Offline functionality
